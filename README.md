# Dynamic Greetings 🌟

Welcome to **Dynamic Greetings**! This is a fun and interactive web application that displays personalized greetings based on the time of day. Whether it's morning, afternoon, evening, or night, get ready to be greeted with style! ☀️🌅🌙🌃

## Features ✨
- **Time-Based Greetings**: Automatically detects the current time and shows appropriate greetings.
- **Responsive Design**: Looks great on all devices, from desktops to mobiles.
- **Customizable**: Easy to modify and extend with your own greetings.
- **Lightweight**: Built with pure HTML, CSS, and JavaScript – no heavy frameworks needed.

## Live Demo 🚀
Check out the live demo here: [https://itsiamdev.github.io/Dynamic-Greetings/](https://itsiamdev.github.io/Dynamic-Greetings/)

## How to Use 📖
1. Clone the repository.
2. Open `index.html` in your favorite web browser.
3. Enjoy the dynamic greetings!

## Technologies Used 🛠️
- HTML5
- CSS3
- JavaScript (Vanilla)
